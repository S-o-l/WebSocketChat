About this project

This is a demo of simple Websocket chat made to qualify knowledge of software development, implementation of modern front-end and back-end technologies and web-design.

This demo uses CSS framework Twitter Bootstrap, Javascript and JQuery for front-end and WebSockets with Jetty server for back-end

This project was done basing on course by Nikolay Tkachenko and his training center Javanec.kiev.ua that took place on November - December, 2012.
